## 🌟 About This Bot

Links Share Bot is a Telegram bot that generates secure, temporary invite links for private channels and manages multi-channel access safely.

Built with Pyrogram + MongoDB, it supports multi-channel management, force subscription, admin control, and bulk link generation.

## 🚀 Features

- Multi-channel support  
- Secure auto-expiring invite links  
- Bulk link generation  
- Force subscription system  
- Request link system  
- Admin control panel  
- Channel management tools  
- User statistics system  

## 🛠 Commands

### Channel Management
- /addch <channel_id>
- /delch <channel_id>
- /channels
- /links
- /bulklink <ids>

### Request System
- /reqlink
- /reqtime
- /reqmode
- /approveon
- /approveoff

### Admin Commands
- /addadmin
- /broadcast
- /stats

## ⚙️ Deployment

### VPS
git clone https://github.com/Xzrie/LinksShareBot
cd LinksShareBot
pip install -r requirements.txt
python3 main.py

### Docker
docker build -t linksharebot .
docker run -d --name linksharebot --env-file .env --restart always linksharebot
